# QuoBlog 

QuoBlog AI Writer is an intelligent blogging assistant that automatically generates **high-quality blog descriptions** using AI. It helps writers quickly create engaging content, summaries, and descriptions with minimal effort.

---

## ✨ Features

* 🤖 AI-generated blog descriptions
* 📝 Automatic content summarization
* ⚡ Fast response generation
* 🎯 SEO-friendly descriptions
* ✍️ One-click content generation
* 🔄 Regenerate description option
* 📡 API-based AI integration
* 🧠 Smart prompt handling
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React 19
* Tailwind CSS
* Axios
* React Hook Form
* React Toastify
* Quill Editor

### Backend

* Node.js
* Express.js
* MongoDB
* Google GenAI
* JWT Authentication

---

## 🚀 How It Works

1. User enters blog **title**
2. Clicks **Generate Description**
3. AI processes the prompt
4. Description is generated
5. User can edit or publish
